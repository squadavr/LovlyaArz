После активации скрипт начинает спамить Alt + Enter для того чтобы словить лавку. 
Для деактивации скрипта нажмите INSERT(лучше проспамить)

Скриптті іске қосу үшін /lovla деп жазыңыз.
Іске қосылғаннан кейін, скрипт дүкенді (лавканы) іліп алу үшін Alt + Enter пернелерін үнемі баса бастайды (спамдайды).
Скриптті өшіру үшін INSERT пернесін басыңыз (бірнеше рет басып жіберген дұрысырақ).
